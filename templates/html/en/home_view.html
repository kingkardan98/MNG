Welcome to your personal online wallet: MNNG!

With MNNG, you will be able to keep track of your expenses in an effortless manner.

This is a service that relies on personal agency, so be honest with yourself!

To start using MNNG, first create a new user.

If you already have an account, login!

Once you've done that, create a member associated with that acocunt.

This will keep track of your economic availability.

You can set an expense limit, so you always know your goals.

A user can be associated with multiple members, so multiple accounts (or multiple people) can be tracked using the same user.

Start using MNNG today!
